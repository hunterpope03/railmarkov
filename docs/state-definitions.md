## Railcar State Definitions

* **in transit**: departed + arrived

* **to yard**: arrived + ... + set out

* **in yard**: set out + classified

* **under inspection**: inspection started + inspection finished

* **defective**: bad order + repair started

* **under repair**: repair started + repair finished

* **from yard**: pick up + ... departed

* **to industry**: arrived + ... + spotted

* **loading**: load started + load finished

* **unloading**: unload started + unload finished

* **from industry**: released + ... + departed
